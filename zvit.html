Звіти з лабораторних робіт

З дисципліни "Інтернет-технології та проєктування WEB-застосувань"

Студента групи ІС-12 Бабича Михайла
Галузь: Туристичні послуги.
Тема: Cервіс для створення та вибору туристичних маршрутів .

Функціонал:

  • Пошук маршрутів по містам
  • Вибір найпопулярніших маршрутів у містах
  • Можливість увійти в обліковий запис
  • Можливість змінити пароль
  • Можливість вийти з облікового запису
  • Можливість видалити профіль

Основні сторінки веб-сайту:

  • Головна(лендінг) містить інформацію про проект, посилання на популярні маршрути та кнопки для реєстрації/входу в обліковий запис
  • Сторінка створення маршруту
  • Сторінка реєстрації
  • Сторінка для входу в обліковий запис
  • Сторінка для зміни паролю
  • Сторінка для виходу з облікового запису
  • Сторінка для видалення профілю
  • Сторінка зі списком останніх маршрутів
  • Сторінка зі списком найпопулярніших маршрутів
  • Сторінка зі списком власних маршрутів
  • Сторінка зі списком маршрутів якими поділилися з користувачем
  • Сторінка зі списком збережених маршрутів
Тема: СТРУКТУРА HTML-ДОКУМЕНТА. ВИБІР ПРЕДМЕТНОЇ ГАЛУЗІ. РОБОТА З ПОСИЛАННЯМИ, ТАБЛИЦЯМИ, ЗОБРАЖЕННЯМИ, СПИСКАМИ в HTML-ДОКУМЕНТІ.
Мета: придбати практичні навички роботи з HTML-документом, таблицями, зображеннями, посиланнями, списками, формами. Створити шаблон звітного HTML-документом для відображення результатів роботи всіх лабораторних робіт.
Місце розташування: Github repository

HTML-документ містить:

  • Head (тут підключено bootstrap)

  • Блок з фоновим зображенням та посиланнями на форму реєстрації
  • Таблицю з прикладами найпопулярніших маршрутів
